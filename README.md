# calculator-team-project
Calculator team project created to gain skills in collaborating on a project using Git. Learn how to create branches, work with pull requests, and resolve merge conflicts.
Team Lead creates a repository, other team members must implement the methods assigned to them, working in a separate branch.
After finishing working on the method, a pull request is created to merge the code into the main branch.
The Team Lead reviews and approves pull requests.
After completing all tasks, local copy of the repository is updated and the application is tested.
